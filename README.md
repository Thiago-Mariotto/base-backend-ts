## What's it?

This is a generator of folders and initial files for developing a project using node.

It is possible to define whether or not to use a database with an ORM. Work with Typescript or just JavaScript. Start all ESLinter configuration. In addition to generating the initial structure of folders to work with an MSC (Model, Service, Controller) architecture, also known as a layered architecture.

## How to use

Clone this repository

```sh
$ git@github.com:Thiago-Mariotto/base-backend-ts.git
```
Add permission for the installation file.
```sh
$ chmod +x install.sh
```

Execute the `install.sh`
```sh
$ ./install.sh
```

Follow the instructions to initialize project.
